# Secured-Notes-App
A full-stack secured Notes Application built using Spring Boot, Spring Security, and React.
The backend provides a RESTful API with advanced authentication and authorization mechanisms, including role-based access control and stateless security.
It covers key concepts such as Security Filter Chain, session management, and secure endpoint protection. 
The frontend is developed with React to deliver a dynamic and user-friendly interface. 
The project follows clean architecture principles and best practices for building scalable and secure modern web applications.
